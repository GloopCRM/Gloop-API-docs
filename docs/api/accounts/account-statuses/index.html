סטטוסי לקוח

באמצעות סטטוסי לקוחות ניתן לסגמנט את הרשומות לפי המצב הרלוונטי של הלקוח.

למה אני צריך לשים לב?#

  1. בResponse תקבלו פרמטר is_active - המשמעות שלו היא האם הסטטוס הזה יהיה מוצג במערכת או שהוא יוסתר ולא ניתן יהיה לבחור בו.

Get Account Statuses#

Endpoint:
Method: GET

GET: https://{{domain}}/v2/accounts/statuses

Authorization: OAuth 2.0
Headers: Authorization: Bearer {{token}}

Body: Content-Type: application/json

Example:

Request - Status Code: 200 OK
